# learnHSK Design System

## Theme
Clean, modern edtech. Bold use of red and gold (Chinese cultural palette). Dark sidebar, light content area.

## Typography
- Headings: Poppins (bold, confident)
- Body: Nunito (friendly, readable)

## Key UI Patterns
- Flashcard flip animation (front: Chinese character + pinyin, back: English + example sentence)
- HSK level selector (HSK 1 through HSK 6+)
- Streak counter with fire emoji in navbar
- Audio play button on each card (TTS or pre-recorded)
- Progress bar per level
- Correct/wrong feedback with color flash (green/red)
- Daily goal ring or progress indicator
